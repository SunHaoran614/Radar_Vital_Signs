# 面向跨域场景的毫米波雷达生命体征检测关键技术研究 - 完整研究方案

## 📋 研究目标

解决毫米波雷达生命体征检测中的**跨域场景泛化问题**，使模型能够在不同雷达系统、不同环境、不同应用场景下保持稳定性能，实现面向跨域场景的关键技术突破。

---

## 🎯 核心问题

### 问题定义

**域偏移（Domain Shift）**：
- 在数据集A（如4TU）上训练的模型
- 在数据集B（如PhysDrive或BGT60TR13C）上性能大幅下降

**原因分析**：
1. **硬件差异**：
   - 雷达频率不同（77-81 GHz vs 60-64 GHz）
   - 雷达芯片不同（AWR1642 vs BGT60TR13C）
   - 天线配置不同（4 RX vs 3 RX）
2. **环境差异**：
   - 室内布局、反射特性不同
   - 测试距离不同
   - 环境噪声水平不同
3. **数据分布差异**：
   - 信号强度、噪声水平不同
   - 数据时长差异（60秒 vs 数分钟 vs 2小时）
   - 受试者群体差异

### 可用数据集

本项目拥有**三个数据集**，为跨域研究提供了丰富的实验场景：

| 数据集 | 雷达型号 | 频率 | 受试者 | 时长 | 数据格式 | 特点 |
|--------|---------|------|--------|------|----------|------|
| **4TU** | AWR1642 | 77-81 GHz | 10人 | 60秒 | 原始ADC | 极端生理场景 |
| **PhysDrive** | 未公开 | 60 GHz | 48人 | 约30秒/样本 | 已处理RDA | 车载环境、多样化驾驶条件 |
| **BGT60TR13C** | BGT60TR13C | 60-64 GHz | 8人 | 5分钟-2小时 | 原始ADC | 长时监测 |

**数据集详细对比**：

| 特征 | 4TU | PhysDrive | BGT60TR13C |
|------|-----|-----------|------------|
| **数据形状** | (4, 250, 153600) | (600, 2, 8, 16, 8) | 待确认 |
| **维度含义** | (RX, ADC, Chirps) | (Time, Re/Im, Doppler, Angle, Range) | 待确认 |
| **环境** | 实验室 | 车载 | 实验室 |
| **场景** | 距离/角度/方向/高度 | 路段/时间/天气/运动 | 长时连续 |
| **干扰** | 受控 | 运动/说话/振动 | 受控 |
| **光照** | 受控 | 自然光 | 受控 |
| **会话数** | 160 (10人×4场景×4重复) | 48会话×多样本 | 待确认 |

**跨域研究场景**：
1. **跨雷达平台**: AWR1642 ↔ BGT60TR13C ↔ PhysDrive雷达
2. **跨频段**: 77-81 GHz ↔ 60-64 GHz ↔ 60 GHz
3. **跨环境**: 实验室(4TU/BGT60) ↔ 车载(PhysDrive)
4. **跨时长**: 短时(30秒-60秒) ↔ 长时(2小时)
5. **跨干扰**: 静态受控 ↔ 动态真实场景
6. **跨数据格式**: 原始ADC ↔ 已处理RDA

## 📊 评估指标

### 主要指标

1. **MAE (Mean Absolute Error)**
   ```
   MAE = (1/N) Σ |y_pred - y_true|
   ```

2. **RMSE (Root Mean Square Error)**
   ```
   RMSE = sqrt((1/N) Σ (y_pred - y_true)²)
   ```

3. **相关系数 (Correlation)**
   ```
   r = corr(y_pred, y_true)
   ```

4. **准确率 (Accuracy within ±5 bpm)**
   ```
   Acc = (预测误差≤5bpm的样本数) / 总样本数
   ```

### 域适应特定指标

1. **域距离 (Domain Distance)**
   - MMD距离
   - A-distance

2. **域分类准确率**
   - 越低越好（说明域不变特征学得好）
