# Radar Vital Signs - 雷达生命体征检测研究项目

## 项目简介

本项目专注于雷达生命体征检测领域的创新研究，基于多个公开数据集和自采集数据，探索跨数据集泛化和极端场景处理等前沿问题。

## 核心研究方向

### 🎯 1. 跨数据集泛化研究
**研究目标**：解决雷达生命体征检测在实际部署中的域适应问题

**研究内容**：
- **域适应与迁移学习**：研究不同雷达硬件、不同采集环境之间的域偏移问题
- **跨场景鲁棒性**：开发能够适应多种场景（实验室环境、车载环境、真实应用场景）的统一算法框架
- **少样本学习**：利用公开数据集的大量标注数据，实现在新场景下的快速适应
- **模型泛化能力评估**：建立跨数据集性能评估体系

**研究价值**：
- 提高算法在实际部署时的泛化能力和鲁棒性
- 降低新场景下的数据采集和标注成本
- 推动雷达生命体征检测技术的实用化进程

### 🔬 2. 极端场景处理研究
**研究目标**：利用4TU数据集的独特数据，开发能够处理极端生理状态的算法

**研究内容**：
- **极端生理状态监测**：
  - 运动后高心率状态（120-150 bpm）的准确检测
  - 哮喘患者的异常呼吸模式识别
  - 冥想状态下的低心率和特殊呼吸模式分析

- **宽范围生理参数估计**：
  - 设计能够覆盖更宽心率范围（40-150 bpm）的算法
  - 适应不同生理状态下的信号特征变化

- **特殊人群适配**：
  - 研究不同生理特征人群的信号差异
  - 开发个性化的信号处理策略

**研究价值**：
- 扩展雷达生命体征监测的应用范围
- 提高算法在复杂生理状态下的准确性
- 为医疗监护、运动健康等应用提供技术支撑

## 数据集

项目使用以下公开数据集：

### 1. **4TU.ResearchD** - 极端生理场景毫米波雷达数据集
- **雷达型号**: AWR1642BOOST (Texas Instruments)
- **工作频率**: 60 GHz
- **参与者**: 10人
- **特色**: 包含运动后、哮喘患者、冥想练习者等极端生理场景
- **测试场景**: 距离、角度、方向、运动后
- **参考设备**: Polar H10 心率传感器

### 2. **PhysDrive** - 车载驾驶员多模态生理监测数据集
- **场景数**: 48个受试者-场景组合
- **传感器**: 毫米波雷达 + RGB + IR + 生理传感器
- **特色**: 真实车载驾驶环境，不同天气和时间条件
- **应用**: 驾驶员疲劳检测、健康监测

### 3. **自采集数据集**
- 用于验证算法的泛化能力和实际应用效果

详细数据集说明请参考 [数据集文档](docs/DATASETS.md)

## 研究方法

### 实验设计

#### 实验1：基线性能评估
- 在每个数据集上单独训练和测试
- 建立性能基线
- 分析各数据集的特点和难点

#### 实验2：跨数据集泛化测试
- **训练集**: 4TU.ResearchD / PhysDrive
- **测试集**: 其他数据集（包括自采集数据）
- **目标**: 分析性能下降原因，识别域偏移的主要因素

#### 实验3：迁移学习实验
- 在公开数据集上预训练
- 在目标数据集上微调
- 对比不同迁移策略的效果

#### 实验4：极端场景鲁棒性测试
- 利用4TU数据集的极端生理状态数据
- 测试算法在宽心率范围下的性能
- 验证特殊人群的适用性

### 评估指标

- **心率估计**: MAE (平均绝对误差), RMSE (均方根误差), 相关系数
- **呼吸率估计**: MAE, RMSE, 相关系数
- **跨数据集性能**: 性能下降率, 域适应效果
- **极端场景**: 不同心率范围下的准确率
