# 毫米波雷达生命体征检测关键技术研究

> **硕士学位论文**: 面向跨域场景的毫米波雷达生命体征检测关键技术研究

## 项目简介

本项目是一个系统性的毫米波雷达生命体征检测研究项目，基于两个公开数据集（4TU.ResearchD、PhysDrive）和一个自采集数据集（BGT60TR13C_Indoor），从**信号处理算法**、**深度学习模型**、**跨域泛化**、**极端场景处理**等多个维度展开研究，旨在突破毫米波雷达生命体征检测的关键技术瓶颈，形成一套完整的理论方法体系和实用算法框架。

## 核心研究方向

### 🔬 1. 信号处理算法创新
**研究目标**：针对毫米波雷达生命体征信号的特点，设计高效、鲁棒的信号处理流程

**研究内容**：
- **雷达信号预处理优化**：
  - 静态杂波抑制算法（背景消除、高通滤波）
  - 多通道信号融合策略（相位域、幅度域）
  - 自适应噪声抑制技术

- **生命体征信号提取**：
  - 改进的相位解缠算法
  - 基于小波变换的多尺度分析
  - 自适应带通滤波器设计
  - 心率呼吸分离算法

- **频谱分析与峰值检测**：
  - 高分辨率频谱估计（FFT、MUSIC、Welch）
  - 自适应峰值检测算法
  - 谐波干扰抑制技术

**创新点**：
- 提出适用于不同雷达平台的统一信号处理框架
- 设计自适应参数调整机制，提高算法鲁棒性
- 优化计算效率，支持实时处理

### 🧠 2. 深度学习模型设计
**研究目标**：利用深度学习技术提升生命体征检测的准确性和泛化能力

**研究内容**：
- **端到端深度学习模型**：
  - 基于CNN的雷达信号特征提取
  - 基于RNN/LSTM的时序建模
  - 基于Transformer的注意力机制
  - 多任务学习框架（心率+呼吸联合估计）

- **模型架构创新**：
  - 轻量化网络设计（适用于边缘设备）
  - 多尺度特征融合
  - 残差连接与跳跃连接
  - 自注意力机制

- **损失函数设计**：
  - 多目标联合优化
  - 不确定性建模
  - 对抗训练策略

**创新点**：
- 设计专门针对雷达信号特征的网络架构
- 提出多模态融合策略（雷达+其他传感器）
- 实现模型可解释性分析

### 🎯 3. 跨域场景泛化研究
**研究目标**：解决雷达生命体征检测在不同域之间的泛化问题

**研究内容**：
- **域适应与迁移学习**：
  - 跨雷达平台适应（AWR1642 ↔ BGT60TR13C）
  - 跨频段适应（60 GHz ↔ 77 GHz）
  - 跨环境适应（实验室 → 车载 → 真实场景）
  - 跨时长适应（短时 → 长时监测）

- **域不变特征学习**：
  - 对抗域适应（DANN）
  - 最大均值差异（MMD）
  - 域混淆技术
  - 元学习方法

- **少样本学习**：
  - 利用公开数据集预训练
  - 目标域快速微调
  - 数据增强策略

**创新点**：
- 建立跨雷达平台、跨环境、跨时长的统一泛化框架
- 提出适用于雷达信号的域适应方法
- 降低新场景部署成本

### 🚀 4. 极端场景鲁棒性研究
**研究目标**：提升算法在极端生理状态和复杂场景下的性能

**研究内容**：
- **极端生理状态处理**：
  - 高心率场景（运动后 120-150 bpm）
  - 低心率场景（冥想状态 40-60 bpm）
  - 异常呼吸模式（哮喘患者）
  - 宽范围生理参数估计

- **复杂环境适应**：
  - 运动干扰抑制
  - 多目标场景处理
  - 低信噪比信号增强

- **长时稳定性**：
  - 长时连续监测（2小时+）
  - 时变参数自适应
  - 漂移补偿技术

**创新点**：
- 设计覆盖宽范围生理参数的算法
- 提出极端场景下的鲁棒性增强策略
- 实现长时稳定监测

## 数据集

项目使用三个数据集，包括两个公开数据集和一个自采集数据集：

### 1. **4TU.ResearchD** - 极端生理场景毫米波雷达数据集（公开）
- **雷达型号**: AWR1642BOOST (Texas Instruments)
- **工作频率**: 77-81 GHz
- **参与者**: 10人
- **特色**: 包含运动后、哮喘患者、冥想练习者等极端生理场景
- **测试场景**: 距离、角度、方向、运动后
- **参考设备**: Polar H10 心率传感器
- **数据时长**: 每次测量60秒

### 2. **PhysDrive** - 车载驾驶员多模态生理监测数据集（公开）
- **场景数**: 48个受试者-场景组合
- **传感器**: 毫米波雷达 + RGB + IR + 生理传感器
- **特色**: 真实车载驾驶环境，不同天气和时间条件
- **应用**: 驾驶员疲劳检测、健康监测
- **数据时长**: 变化，通常数分钟

### 3. **BGT60TR13C_Indoor** - 室内长时连续监测数据集（自采集）
- **雷达型号**: BGT60TR13C (Infineon Technologies)
- **工作频率**: 60-64 GHz
- **参与者**: 8人
- **特色**:
  - **长时监测**: 包含1名受试者的2小时连续监测数据
  - **多受试者**: 8名受试者的短时测量（5-10分钟/人）
  - **室内环境**: 真实室内场景，距离80-100 cm
- **参考设备**: Bigrun心率带（蓝牙心率监测）
- **应用**: 长期稳定性研究、跨雷达平台验证

**数据集协同价值**：
- **跨数据集泛化**: 4TU → PhysDrive → BGT60TR13C
- **跨雷达平台**: AWR1642 (77-81 GHz) → BGT60TR13C (60-64 GHz)
- **跨场景**: 极端生理 → 车载 → 室内静态
- **跨时长**: 短时（60秒）→ 中时（数分钟）→ 长时（2小时）

详细数据集说明请参考 [数据集文档](docs/DATASETS.md)

## 技术路线与实验设计

### 研究方法体系

```
信号处理层 → 特征提取层 → 模型预测层 → 域适应层 → 应用层
    ↓            ↓            ↓            ↓         ↓
  预处理      传统方法      深度学习      泛化增强    实际部署
  杂波抑制    频谱分析      CNN/RNN      域适应      边缘计算
  信号融合    峰值检测      Transformer   迁移学习    实时监测
```

### 实验设计

#### 实验组1：信号处理算法对比实验
- **目标**: 评估不同信号处理方法的性能
- **内容**:
  - 对比不同杂波抑制算法（背景消除、高通滤波、ICA）
  - 对比不同相位提取方法（单通道、多通道融合）
  - 对比不同频谱估计方法（FFT、Welch、MUSIC）
  - 对比不同峰值检测策略
- **数据集**: 三个数据集全覆盖
- **指标**: MAE, RMSE, 相关系数, 计算时间

#### 实验组2：深度学习模型对比实验
- **目标**: 设计并评估不同深度学习架构
- **内容**:
  - 基线模型：传统信号处理方法
  - CNN模型：1D-CNN, 2D-CNN, ResNet
  - RNN模型：LSTM, GRU, BiLSTM
  - Transformer模型：自注意力机制
  - 混合模型：CNN+LSTM, CNN+Transformer
- **训练策略**: 端到端训练 vs 特征提取+分类
- **数据集**: 单数据集训练与测试
- **指标**: 准确率、泛化能力、计算复杂度

#### 实验组3：跨域泛化实验
- **目标**: 评估跨域场景下的性能与改进方法
- **内容**:
  - **基线测试**: 源域训练 → 目标域测试（无适应）
  - **迁移学习**: 预训练 + 微调
  - **域适应**: DANN, MMD, 对抗训练
  - **元学习**: MAML, Prototypical Networks
- **跨域场景**:
  - 4TU → PhysDrive → BGT60TR13C
  - AWR1642 → BGT60TR13C（跨雷达平台）
  - 实验室 → 车载 → 真实场景（跨环境）
- **指标**: 性能下降率, 域适应增益, 少样本学习效果

#### 实验组4：极端场景鲁棒性实验
- **目标**: 验证算法在极端条件下的性能
- **内容**:
  - 高心率场景（运动后 120-150 bpm）
  - 低心率场景（冥想 40-60 bpm）
  - 异常呼吸（哮喘患者）
  - 低信噪比场景
  - 运动干扰场景
- **数据集**: 主要使用4TU数据集
- **指标**: 不同心率范围的准确率, 鲁棒性指标

#### 实验组5：长时稳定性实验
- **目标**: 评估长时连续监测的稳定性
- **内容**:
  - 2小时连续监测数据分析
  - 时变参数自适应
  - 漂移检测与补偿
- **数据集**: BGT60TR13C长时数据
- **指标**: 长时MAE, 稳定性指标, 漂移率

#### 实验组6：消融实验
- **目标**: 分析各模块的贡献
- **内容**:
  - 信号处理模块消融
  - 模型组件消融（注意力机制、残差连接等）
  - 域适应模块消融
  - 数据增强策略消融
- **指标**: 各模块的性能增益

### 评估指标体系

#### 基础性能指标
- **心率估计**: MAE, RMSE, 相关系数, 准确率（±5 bpm）
- **呼吸率估计**: MAE, RMSE, 相关系数, 准确率（±2 rpm）

#### 泛化性能指标
- **跨域性能**: 性能下降率, 域适应增益
- **少样本性能**: 不同样本数下的准确率曲线

#### 鲁棒性指标
- **极端场景**: 不同心率/呼吸率范围的准确率
- **噪声鲁棒性**: 不同SNR下的性能
- **长时稳定性**: 长时MAE, 漂移率

#### 计算效率指标
- **推理时间**: 单样本处理时间
- **模型大小**: 参数量, 存储空间
- **能耗**: 适用于边缘设备部署
